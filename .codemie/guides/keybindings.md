# keybindigns.md

The following is an agreed list of default keybindings for `shame`.
If the required keybinding is not listed here, follow the principle of least surprise and propose vim-like bindings where possible.
| Key | Action |
|---|---|
| `j` / `k` | Move cursor down / up |
| `ctrl-d` / `ctrl-u` | Half page down / up |
| `g` / `G` | First / last line |
| `,` | Navigate to parent commit (re-blame) |
| `<` | Go back in blame history |
| `d` | Open full-screen diff for current line's commit |
| `Enter` | Open commit info (changed files + diff preview) |
| `q` | Quit (from blame) / return to blame (from diff) |
| `/` | Start code search |
| `n` / `N` | Next / previous search match |
| `c` | Open column toggle menu |
| `left`/`h`, `right`/`l` | Horizontal scroll |
